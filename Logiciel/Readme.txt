_______READ ME_______

"Logiciel de génération de fractales"

Lire la documentation préalablement.

1. Pour lancer le programme, double cliquez sur l'executable.

Si celui-ci n'existe pas, ouvrez un Terminal (ou Invite de commandes)
Randez-vous dans le dossier racine du programme :
	~/Logiciel
Une fois dans ce dossier, lancer la commande :
	make all
Atendez que la compilation se termine.

Vous pourrez lancer le logiciel à partir du Terminal avec la commande :
	./logiciel
Ou en faisant (1).
